# Agente_RAG

Este repositorio contiene un agente conversacional de tipo RAG desarrollado en Botpress

Funciones:

* Realiza busquedas de contenido
* Responde deacuerdo con el contenido
* Acepta instrucciones especializadas
* Utiliza GPT4-o mini
